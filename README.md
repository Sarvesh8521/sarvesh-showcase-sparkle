# Sarvesh Singh — Portfolio

Personal portfolio site for Sarvesh Singh, Python Backend Developer.

## Tech stack

- Vite + React + TypeScript
- Tailwind CSS + shadcn/ui
- Supabase (auth, database, storage) for the admin dashboard

## Local development

```sh
npm install
npm run dev
```

## Admin

Visit `/admin` to sign in and update the profile photo, resume, and bio.
